# 🧠 AI Lead Management System for Clinics

Sistema inteligente para automatizar la gestión de pacientes en clínicas de fisioterapia mediante IA.

## 🚀 Problema

Muchas clínicas reciben mensajes de pacientes por distintos canales y no tienen un sistema eficiente para:

- Responder rápidamente
- Clasificar consultas
- Gestionar citas
- Hacer seguimiento de pacientes

Esto provoca pérdida de clientes y desorganización.

---

## 💡 Solución

Este proyecto implementa un asistente inteligente que:

- Responde automáticamente a los pacientes usando IA
- Clasifica los mensajes (cita, dolor, otros)
- Guarda los leads en CSV y Google Sheets
- Permite visualizar y gestionar los leads desde una app web
- Automatiza el agendado de citas con Google Calendar
- Envía confirmaciones por email

---

## 🛠️ Tecnologías utilizadas

- Python
- Streamlit
- Claude (Anthropic API)
- Google Sheets API
- Google Calendar API
- SMTP (email)
- Pandas

---

## 📋 Funcionalidades

- Chat interactivo con pacientes
- Clasificación automática de mensajes
- Registro de leads
- Dashboard con métricas
- Filtros y visualización de datos
- Gestión de estados (nuevo, contactado, cita agendada)
- Agendado automático de citas
- Envío de email de confirmación
